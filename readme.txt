== Installation ==
1. Upload the \'smart-author-widget.zip\' folder to the \'/wp-content/plugins/\' directory.
2. Activate the \'Author Bio Widget\' plugin through the \'Plugins\' menu in WordPress.
3. Add and set Widget from Appearance->Widget 

== Frequently Asked Questions ==
How to use Metro Author Widget
*Go to Appearance > Widgets
*Choose Metro Author Widget and either drag it to the sidebar where you wish it to appear, or click the widget, (select a destination sidebar if your theme has more than one) and click the Add Widget button.
